
-- SUMMARY --

Include soundninja js on enabled content types.


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.



-- CONFIGURATION --

* Configure user permissions in Administration » People » Permissions:

  - Access Soundninja Configuration

    Users in roles with the "Access Soundninja Configuration" permission will see
    configuration and checkbox on the node form.
